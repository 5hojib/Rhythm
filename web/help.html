Help & How to Use Rhythm

Learn how to get the most out of Rhythm music player with our comprehensive guide.

Getting Started Playing Music Library Customization Settings Troubleshooting Support

Getting Started

Installation & Requirements

Download Rhythm from our GitHub releases or F-Droid. The app requires Android 8.0+ (API 26), 2GB RAM minimum, and 50MB storage. Enable "Install from unknown sources" for APK installation.

9-Step Setup Process

First launch guides you through permissions, theming setup, and media scanning. Grant storage permissions to access your music library. The app will automatically scan for music files.

Essential Permissions

  • Storage Access: Scan and play music files from device storage
  • Notifications: Playback controls, updates, and media session
  • Bluetooth: Wireless audio device support
  • Internet: Lyrics (LRCLib), artwork (Deezer), and app updates

Material You & Theming

Rhythm supports Android 12+ dynamic theming with colors extracted from your wallpaper. Choose between light/dark modes or follow system settings for seamless adaptation.

Playing Music

Professional Audio Engine

Rhythm uses Media3 ExoPlayer for studio-quality playback with gapless transitions. Supports FLAC, ALAC, MP3, AAC, OGG, WAV, and more formats.

Smart Playback Features

  • Auto-Resume: Continues playback exactly where you left off
  • Smart Audio Focus: Handles calls and notifications gracefully
  • Canvas Support: Dynamic Spotify Canvas visuals for compatible tracks
  • Advanced Shuffle: True random, artist-aware, and weighted algorithms

Synchronized Lyrics

Real-time lyrics integration with LRCLib. Tap timestamps to seek, auto-scroll during playback. Manual reload available if lyrics fail to load.

Queue & Playback Control

Long-press songs to add to queue. Reorder tracks, remove items, or clear entire queue. Independent volume control or system integration.

Equalizer & Audio Customization

Built-in 5-band equalizer with presets. Adjust bass, treble, and audio settings. Save custom equalizer profiles.

Library Management

Smart Library Organization

Navigate through Songs, Albums, Artists, Playlists, and Folders tabs. Customize tab order in Settings → Library to match your preferences.

Lightning-Fast Search

Fuzzy search across your entire library. Search by song title, artist, album, or lyrics. Results update in real-time as you type.

Advanced Media Scanning

  • Automatic Scanning: Scans device on first launch and app updates
  • Whitelist Mode: Include only specific folders
  • Blacklist Mode: Exclude unwanted directories
  • Explorer: Manually browse and add music folders

Playlist Management

Create, edit, and manage playlists. Import/export in various formats. Add songs via long-press or drag-and-drop. Smart playlist suggestions.

Rich Metadata & Artwork

Automatic high-quality album art fetching from Deezer. Comprehensive tag support with embedded metadata display.

Customization

Themes & Appearance

Choose between light/dark themes or follow system settings. Enable Material You for dynamic colors based on your wallpaper.

Color Themes

Extract colors from album artwork for dynamic theming, or choose from predefined color schemes in Settings.

Fonts

Change the app's font from system fonts or import custom fonts. Access this option in Settings → Appearance.

Tab Order

Reorder the navigation tabs (Songs, Albums, Artists, etc.) to match your preferences in Settings → Library.

Settings & Advanced Features

Smart Settings Search

Use the search bar in Settings to instantly find any option. Type keywords to filter through all available settings and preferences.

Backup & Restore

Backup playlists, settings, and preferences. Restore them when reinstalling or switching devices. Includes theme preferences and equalizer settings.

Sleep Timer & Auto-Stop

Set timers to automatically stop playback after a specified duration. Choose from presets or custom time. Visual countdown indicator.

API Management

Configure API endpoints for lyrics (LRCLib) and artwork (Deezer). Set custom API keys or use default community services.

Cache Management

Control artwork, lyrics, and Canvas cache sizes. Clear cache to free up space or refresh content. Automatic cache cleanup options.

Auto-Updates

Automatic update checking via GitHub releases. Download and install updates directly within the app. Beta program access for early features.

Device Output & Bluetooth

Advanced audio output selection. Automatic Bluetooth device detection and switching. Audio focus management for calls and notifications.

Troubleshooting

Music Not Showing in Library

Rescan media in Settings → Library. Check whitelist/blacklist settings. Use Explorer to manually add folders. Ensure files aren't corrupted or in unsupported formats.

Permission & Storage Issues

Go to Settings → Apps → Rhythm → Permissions and grant storage access manually. Restart app after granting permissions. Check if files are in accessible locations.

Playback Problems

  • Audio Glitches: Clear cache in Settings → Advanced → Cache Management
  • No Sound: Check Device Output settings and Bluetooth connections
  • Gapless Issues: Ensure tracks are in compatible formats (FLAC, ALAC preferred)

Lyrics Not Loading

Check internet connection and API settings. Try manual reload in lyrics view. Some tracks may not have lyrics available in LRCLib database.

Artwork Not Showing

Clear artwork cache in Settings → Advanced. Check Deezer API configuration. Some albums may not have available artwork.

App Performance Issues

Clear all caches, restart app. Check for updates. If issues persist, check crash logs in Settings → Advanced → Crash Logs.

Memory Leaks & ANR Detection

Debug builds include LeakCanary and ANR monitoring. Check logs for memory issues. Restart app if experiencing slowdowns.
